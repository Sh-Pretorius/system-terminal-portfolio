# 🖥 System Terminal Portfolio
An immersive, retro-futuristic developer portfolio built with Next.js (App Router), React, and TypeScript.
Designed as a simulated computer terminal experience featuring boot sequences, CRT screen effects, glitch animations, and modular system panels.

## Live Demo
👉 View Portfolio: https://shania-pretorius.vercel.app/

## Concept
Instead of a traditional portfolio layout, this project recreates a digital operating system interface.
The user experience includes:
- ⚡ Boot sequence simulation
- 📺 CRT monitor frame effects
- 🧠 Diagnostic-style skill panels
- 📡 Animated transmissions
- 🔊 Signal drift & glitch text effects
- ⌨ Typewriter-style text animations
The goal was to merge storytelling with frontend engineering to create a cohesive, immersive experience.

## Tech Stack
- Next.js 13+ (App Router)
- React
- TypeScript
- Custom CSS animations
- Modular component-based architecture

## Architecture
The project follows a layered UI architecture:
- *Layout Layer* – Global wrappers and structural composition
- *Experience Layer* – Boot sequence, CRT effects, glitch system
- *Interface Layer* – System navigation and panels
- *Content Layer* – Identity, capabilities, and projects
This structure keeps effects isolated while maintaining clean component boundaries.

## Key Features
- Immersive UI design with consistent thematic identity
- Reusable system-style panel components
- Animation-driven user experience
- Modular and scalable component structure
- Clean separation of layout, effects, and content
